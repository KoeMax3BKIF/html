Task 1 filter

Finde alle Studierenden, die bestanden haben (note <= 4). Speichere das Ergebnis in passed.

students.filter(s => s.note <= 4) → Behalte nur jene, bei denen die Bedingung true ergibt.

Task 2 map

Erstelle ein Array von Strings im Format "Lukas Huber (28)". Speichere es in labels.

students.map(s => `${s.name} (${s.alter})`) → Template Literal: Backticks mit ${} fur Variablen.

Task 3 filter + map

Extrahiere aus den bestandenen Studierenden (note <= 4) nur deren Namen in ein Array passedNames.

Verkette beide Methoden: students.filter(s => s.note <= 4).map(s => s.name)

Task 4 reduce

Berechne die Durchschnittsnote aller Studierenden. Speichere das Ergebnis in averageGrade.

Erst alle Noten summieren, dann durch die Anzahl teilen: students.reduce((sum, s) => sum + s.note, 0) / students.length

Task 5 – Bonus Chaining

In einer einzigen Kette: Finde die Namen aller Studierenden mit alter >= 17 die bestanden haben (note <= 4), verbunden als kommaseparierten String. Speichere in result.

Mehrere Methoden hintereinander hängen: students .filter(s => s.alter >= 17) .filter(s => s.note <= 4) .map(s => s.name) .join(", ")
